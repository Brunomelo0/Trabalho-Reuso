# Projeto LMS

## Backend:
### Instalar dependências na Pasta API:
```sh
npm install
```
### Rodar API:
```sh
node server.js
```
### Criar Banco de Dados Postgres:
```sh
- Configurar o arquivo \Backend\API\.env com os dados de conexão do pgadmin
- Utilizar no pgadmin o script SQL presente no arquivo:
    \Backend\DataBase\Tables.sql
```
## Frontend:

```sh
- Abrir arquivo:
\Frontend\home.html
```
